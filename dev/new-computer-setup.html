Setting Up a New Windows Computer

Some notes about what I do to setup a new Windows laptop

Windows Update

First things first.. check for updates. Come back tomorrow after all the updates have finished.

Software

Now, on to installing some software. Here's the basic set of software I installed on my most recent new laptop.
  • Linux Subsystem
    • Enable WSL in Windows Features
    • Install Ubuntu or other linux from the Windows Store
  • Chocolatey (or Scoop I suppose)
  • A good broswer (Brave Browser, perhaps)
  • Windows Terminal from the Store
  • Neovim for command line editing
  • VS Code for normal editing
  • Git
  • Node.js (actually nvm)
  • Todoist
  • Slack

Personalization

Let's make this thing feel more comfortable
  • Fonts
    • Cascadia Code
    • Nerd Fonts (IBM Plex Mono NF)
    • Fira Code
  • Wallpaper
    • Currently using a bunch of Jupiter images (from Juno) that change every 30 min
  • Turn on Dark Mode
